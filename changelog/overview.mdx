Stay up to date with the latest changes to Tallysight. This page lists product updates, new features, and bug fixes shipped each week.

<Update label="Week of May 4, 2026" description="In-platform settings, Help menu, and the Developer install page">

## New features

**Settings, Help menu, and Developer install page**

Workspace settings are now available directly in the platform. Open the user menu to find a new **Help** submenu with links to **Documentation** and the **Changelog**, plus a **Developer** section under workspace settings with everything you need to install widgets:

- Copy your embed script and ready-made examples for the Tile, Odds Tile, Bet Finder, and Odds Text widgets.
- Download the latest WordPress plugin with your workspace slug pre-filled and a recommended setup checklist.
- Request API access from the same page — submissions go straight to your account team.

Translations are included for English, Spanish, and Portuguese. See the [quickstart](/quickstart) and [widget installation](/widgets/installation) guides for related setup.

## Updates

**Faster widget loads**

Responses for the [Tile](/widgets/tile), [Odds Text](/widgets/odds-text), and the Odds Tile and Bet Finder widgets are now cached, so repeat requests return faster. Confident matches stay cached longer, and weak matches are rejected so widgets don't render the wrong entity. No configuration changes are required.

## Bug fixes

- **NFL futures coverage on FanDuel and Betfair**: Next-season NFL futures (Super Bowl Winner, AFC and NFC Championship Winner, all eight division winners, AP NFL MVP, DPOY, and CBPOY) now map correctly when sportsbooks publish them with a 2026-27 season suffix.

</Update>

<Update label="Week of May 1, 2026" description="Faster visitor location detection">

## Updates

**Faster visitor location detection**

Visitor location now resolves faster on pages that depend on it, including geolocated [Promo Tiles](/widgets/promo-tiles) offers and the automatic odds format used by the [Tile](/widgets/tile) and [Odds Text](/widgets/odds-text) widgets. No configuration changes are required.

</Update>

<Update label="Week of April 30, 2026" description="Geolocated community offers and longer live-odds windows for baseball and tennis">

## Updates

**Geolocated community offers**

The **Offers** tab on community pages now only shows promos available in the visitor's state and country, so visitors only see offers they can actually claim. Offer titles fall back to the campaign name when one isn't set. See [Promo Tiles](/widgets/promo-tiles) for related promo configuration.

</Update>

<Update label="Week of April 27, 2026" description="Polymarket odds, creator marketplace, and community leaderboards">

## New features

**Polymarket prediction-market odds**

Polymarket prediction-market odds are now available alongside traditional sportsbooks.

**Creator marketplace**

Workspace admins can now surface partner brands to creators, and creators can submit interest in becoming partners. Visitors sign up through the **Become a Creator** modal, and pending interest is automatically claimed once they authenticate. Active partner cards are grouped by category, and admins get a dedicated view of brand requests.

**Community leaderboard page**

Communities now have a dedicated leaderboard page, so members can browse rankings without leaving the community.

## Updates

**Promo type selector**

The promo manager now includes a **Type** dropdown on the Details step, so you can mark a promo as either an **Offer** or a **Link** when creating or editing one. Existing promos are migrated automatically.

**Profile filters refresh instantly**

Sportsbook, status, result, and market filters on the profile page now update betslips immediately, including clearing results when a filter returns no matches.

**Smaller community leaderboard**

The community leaderboard now hides the Money Meter, Avg Winnings, and ROI columns by default to remove horizontal scrolling on smaller screens. The full ranking table on `/rankings` is unchanged.

## Bug fixes

- **Bet builder on storage-constrained sites**: The widget SDK no longer initializes the [bet builder](/widgets/bet-builder) on pages where it's disabled, so embedded sites with limited browser storage no longer see related warnings.
- **Promo manager first load**: The promo list no longer appears empty on first load.
- **Manual pick grading**: Admin manual grades now correctly update leaderboards and rankings.
- **Widget error visibility**: Embedded widgets now surface the actual error when an API call fails instead of an opaque network failure.
- **Live odds for overtime sports**: Live odds for NBA, NFL, NHL, and soccer now stay available longer after scheduled start, so markets aren't closed prematurely on long games.
- **Sportsbook market coverage**: Added and corrected market mappings across BetMGM, BetRivers, Betfair, DraftKings, FanDuel, NorthStar, Sporting Bet, Sports Interaction, and bwin.
- **`/about` page**: The legacy `/about` page now redirects to `tallysight.com/about` so visitors land on a single canonical page.

</Update>
